These are the apps that most designers use to make AR/VR experiences.

Unity

Unity is the most popular interaction engine for designing AR/VR prototypes.

Maya

Most people use Maya to make 3D models that they can import into Unity.

Blender

A free alternative to Maya that's also commonly used to make and rig 3D models.

AfterEffects

Many designers use AfterEffects with Cinema4D to mock up AR concepts.

Cinema4D

Included with AfterEffects. Great for making beautiful mockups and visualizations.

A-Frame

Make VR for the web using HTML and JavaScript.

React VR

VR for the web, but using React.

Lens Studio

Snapchat's desktop authoring tool for creating 3D world filters.

AR Studio

Facebook's desktop authoring tool for Messenger 3D filters.

These are plugins that make AR/VR prototyping possible in other apps.

MRTK

Unity

Microsoft's Mixed Reality Toolkit is focused on HoloLens and Windows Mixed Reality.

ARKit

Unity

Apple's ARKit makes it easier to build AR apps for iPhone in Unity.

ARCore

Unity

Google's ARCore makes it easier to build AR apps in Unity.

Vuforia

Unity

Image and object tracking for AR apps built in Unity. Works with HoloLens.

Crayon

Unity

Web-like UI microinteractions for Unity.

VRTK

Unity

VRTK makes it easy to add cross-platform interaction to VR apps.

Sketch
2VR

Sketch

Quickly preview your 2D Sketch mockups as curved panels in VR.

These are UI component libraries that jumpstart AR/VR prototyping.

Facebook

Sketch & Unity templates from Facebook's VR Design team.

Daydream

Google's Daydream Elements are helpful for jumpstarting Daydream projects.

Some designers are using VR to prototype AR, and use these tools.

Tiltbrush

Great for rapidly sketching 3D concepts.

Blocks

Google's tool for 3D modeling in VR.

Medium

Oculus 3D modeling tool for the Rift.

Modbox

Useful for quickly blocking out 3D elements in space.

Shapes

Remote, collaborative 3D design on High Fidelity's social VR platform.

These are beta tools still in development. You can request access to them.

Torch3D

Prototyping and collaboration in AR/VR for teams & individuals.

wiARframe

Dedicated AR prototyping tool developed by Jeremiah Alexander.
